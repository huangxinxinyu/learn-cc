# 项目宗旨
- 这个项目宗旨是学习claude code的架构
- 当中包括agent交互，工具调用，上下文管理，功能添加，数据结构等等
- 学习路径主要参考 https://github.com/shareAI-lab/learn-claude-code
- 每个文件夹下会做一个小的demo来完成特定模块的学习
- 学习目的，所以尽量避免复杂的实现以及框架的搭建，比如连接数据库、redis等等。能够用脚本模拟流程或者将数据打印到控制台如果能够满足学习需求就足够了

## 交互规则
- 大部分时候用引导性或者反问的方式带领我实现，只有我明确要求你输出代码时才输出代码
